# PriceOps (priceops)

PriceOps is a pricing and packaging infrastructure platform for SaaS companies that enables simulation of pricing strategies, design of packaging tiers, and launch of pricing pages within minutes. The platform provides REST APIs for managing customer information, usage records, pricing plans, entitlements, and monetization rules, along with integrations for payment gateways such as Stripe.

## APIs

### PriceOps API

REST API for managing customer information and usage records within the PriceOps pricing infrastructure platform. Supports retrieval of subscribed customers and their active plans, posting usage records for metered billing, and integrating pricing pages into SaaS applications.

- **Human URL:** [https://docs.priceops.net/documentation/developers/customer-information-and-usage-record-api](https://docs.priceops.net/documentation/developers/customer-information-and-usage-record-api)
- **Base URL:** `https://engine.prod.priceops.net/api`
